# Student Tasks – Todo List

Todo list simple pensée pour les étudiants afin de suivre les tâches de révision et de travaux à rendre.  
L’utilisateur peut ajouter des tâches avec matière + description, les cocher comme terminées et filtrer l’affichage.

## Fonctionnalités

- Ajout de tâches avec **subject** et **task description**.
- Marquer une tâche comme terminée / active.
- Filtres : **All**, **Active**, **Done**.
- Suppression d’une tâche.
- Sauvegarde automatique des tâches avec `localStorage` (les données restent après rechargement).

## Stack technique

- HTML5  
- CSS3 (carte centrale, liste, états visuels done/active).
- JavaScript (DOM, événements, filtres, stockage local).

## Objectif du projet

- Pratiquer la gestion de listes, d’état et de filtres en JavaScript.
- Montrer un exemple concret d’app front‑end interactive pour le portfolio.
- Proposer un outil utile pour organiser les tâches de révision.

## Utilisation

1. Remplir le **Subject** (optionnel) et le **Task** (obligatoire).
2. Cliquer sur **Add task** (ou Enter dans le champ Task) pour ajouter la tâche.
3. Cocher une tâche pour la marquer comme **done**.
4. Utiliser les boutons **All / Active / Done** pour filtrer la liste.
5. Cliquer sur **✕** pour supprimer une tâche.
